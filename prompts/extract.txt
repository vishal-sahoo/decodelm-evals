- bare quiz answers (e.g. "B"),

- text the tutor spoke, or source text the user merely pasted or relayed.

Rules:

  block that anchors it. This is used to verify the learning came from the user, so it
  must appear word-for-word inside a <user> block.
- If the user demonstrated no understanding, return an empty list.

Return ONLY a JSON array, nothing else. Each element must be an object:
  {{"learning": "<the concept, in the user's voice>", "user_quote": "<verbatim substring from a <user> block>"}}
